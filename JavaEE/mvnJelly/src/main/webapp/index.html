Welcome to the Jelly Diamonds project.
To access RESTful web services, type URL in the following form:

http://jelly.diamonds/jellyDiamonds-1.0/api/resource

where /resource has to be replaced by one of the proposed web services:

  • test API:
    • /hello - prints hello message [plain text]
    • /random - returns the random integer between 1 and 10 [plain text]
  • Gem API:
    • /gems/addGem/{type}/{color} - adds a gem to the database.
      Replace {type} and {color} by strings OR use the following form:
http://jelly.diamonds/jellyDiamonds-1.0/api
/gems/addGem//
